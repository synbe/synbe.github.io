### 8. 文件编码
​
在打开文件时，可以指定编码类型，如`'utf-8'`：
​
```python
with open('filename.txt', 'r', encoding='utf-8') as file:
    content = file.read()
```
​
### 9. 使用文件对象的方法
​
除了`read()`和`write()`，文件对象还提供了其他一些有用的方法：
​
- `readline()`：读取下一行，不包括末尾的换行符。
​
  ```python
  with open('filename.txt', 'r') as file:
      line = file.readline()
      print(line)
  ```
​
