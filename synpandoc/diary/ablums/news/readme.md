# 使用说明

###  exe版Diary使用说明：

功能：将./input.md 追加./index.html中。

使用方法：

1,编辑./input.md：支持txt、html格式，其中首行作为标题，如插入图片使用img标签。

2,运行./diary.exe：如无./index.html文件则默认自动生成，可添加head、style等标签。



###  python版Diary使用说明：

功能：将./input.md 追加./index.html中。

使用方法：

1,编辑./input.md：支持md、html格式，其中首行前加#作为标题。

2,运行python ./diary.py：如无./index.html文件则默认自动生成，并生成pdf文件，可添加head、style等标签。
